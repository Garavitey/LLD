The Project: Area Calculator
Build a system that calculates the total area of various shapes.
Requirements:
	1.	Shape Classes: Create a Rectangle class (with width and height) and a Circle class (with radius).
	2.	The Calculator: Create an AreaCalculator class that has a method to calculate the total area of an array (or List) of shapes.
	3.	The Execution: In your Main class, create a few shapes, put them in a list, and print the total area.
